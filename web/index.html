Usage

Use babylon_ros as a JavaScript library in your own applications:

import * as babylon_ros from '@ranchhandrobotics/babylon_ros'; // Parse OpenSCAD customizer const result = babylon_ros.parseOpenSCADCustomizer(scadCode); // Convert to STL const stl = await babylon_ros.convertOpenSCADToSTL({ scadContent: scadCode, filename: 'model.scad', parameterOverrides: { width: 20 } });

Hosting Collections

Build a gallery of OpenSCAD models by hosting SCAD files and using URL parameters:

https://example.com/viewer-openscad.html?scad=https://example.com/models/cube.scad https://example.com/viewer-openscad.html?collection=models/&file=cube

File structure example:
models/cube.scad
models/sphere.scad
models/cylinder.scad
